# LearnPath AI

## Project Overview

LearnPath AI is a web application that helps students and learners generate structured learning roadmaps for different skills such as React, Java, DSA, Full Stack Development, and AI/ML.

The platform provides a clear learning path that helps users understand what to learn and in which order.

---

## Problem Statement

Many students struggle to find a proper roadmap while learning a new technology. They often waste time searching for resources and deciding what topics to study first.

---

## Proposed Solution

LearnPath AI generates personalized learning roadmaps based on the user's learning goal. The system provides a structured sequence of topics, projects, and milestones to guide learners effectively.
